# Редизайн главного экрана: блоки «Сейчас» и «Сегодня»

Обновим `src/routes/index.tsx`, чтобы привести два блока к виду со скриншота. Все остальные блоки (заголовок, «Быстро добавить», «События дня», нижняя навигация) остаются без изменений.

## 1. Новый блок «Сейчас» (вставляется над «Быстро добавить»)

Большая карточка `soft-card` с заголовком «Сейчас» слева сверху и текущим временем `HH:MM` справа сверху. Справа внутри карточки — иллюстрация спящего малыша (`@/assets/baby-moon.png`, увеличенная). Слева — список из 3 строк с круглой иконкой и двумя строками текста:

- 🌙 **Бодрствует / Спит N минут** — вычисляется по последнему завершённому/активному сну. Подпись: `после сна HH:MM — HH:MM` или `начался в HH:MM`.
- 🍼 **Кормление Xч Yм назад** — подпись: `последнее в HH:MM · {ml} мл` (мл если есть в `meta.ml`).
- 👶 **Подгузник Xч Yм назад** — подпись: `последний в HH:MM`.

Если данных нет — строка-заглушка («ещё не было сегодня»). Время обновляется каждые 30 сек (используем уже существующий `now` стейт).

Иллюстрация малыша из шапки переезжает в этот блок; в шапке остаётся только текст и кнопка настроек.

## 2. Редизайн блока «Сегодня» (заменяет текущий `TodayTimeline`)

Карточка `soft-card` с заголовком «Сегодня». Внутри — горизонтальный ряд из 4 равных тайлов (flex, gap-3), каждый со своей категорийной иконкой слева и двумя строками справа:

| Иконка | Значение | Подпись |
|---|---|---|
| 🌙 (purple) | `formatDuration(сон)` | Сон |
| 🍼 (blue)   | количество кормлений | Кормлений |
| 👶 (gold)   | количество подгузников | Подгузников |
| 🌳 (green)  | `formatDuration(прогулка)` | Прогулка |

Используем существующие `activityConfig`, `totalDuration`, `formatDuration` из `tracker-store`. Тайлы — мини `liquid-control` с `glow-soft` в цвет категории. Компонент `TodayTimeline` больше не используется на главной (файл оставляем — он может пригодиться, не удаляем).

## 3. Что НЕ меняется

- Шапка с именем и возрастом (только убираем оттуда картинку малыша).
- Блок «Быстро добавить» с 4 тайлами и центральной кнопкой «+».
- Блок «События дня» (последние 3 события + ссылка «Смотреть все»).
- Все стили в `src/styles.css`, токены, glow-классы.
- Маршрут `/history` и компонент `DayOrbit`.

## Технические детали

- Файл правок: только `src/routes/index.tsx`.
- Новый компонент `NowCard` определяем внутри файла (или как отдельный `src/components/NowCard.tsx` — на ваш выбор; по умолчанию — внутри файла, чтобы не плодить файлы).
- Расчёт «Бодрствует N минут»: берём последний сон с `endedAt` < now и без активного сна — `now - endedAt`. Если активный сон — показываем «Спит N минут» от `startedAt`.
- Для кормления и подгузника берём последнее событие соответствующего типа за сегодня.
- Цвета тайлов «Сегодня» — через существующие токены `--sleep`, `--feed`, `--diaper`, `--walk` (как уже сделано в `QuickTile`).
- `suppressHydrationWarning` на динамических временах.
